隐私政策

Caption Turbo · 最后更新: 2026年6月20日

Caption Turbo 不收集、不存储、不传输任何用户数据。所有音频处理和语音识别均在您的设备本地完成。

数据处理

Caption Turbo 是一款完全离线运行的 macOS 应用程序。当您使用本应用转录音视频文件时，所有处理过程（包括音频提取、语音识别和字幕生成）均在您的 Mac 本地完成。没有任何数据会离开您的设备。

网络连接

本应用不需要互联网连接即可正常运行。本应用不会建立任何网络连接，不会向任何服务器发送数据。

数据存储

本应用仅在您的设备上保存以下内容：

· 应用偏好设置（如语言、帧率、输出目录等），存储在 macOS UserDefaults 中
· 生成的字幕文件（SRT / FCPXML），保存在您选择的输出目录中
· 转录过程中的临时文件，处理完成后自动清理

第三方服务

本应用使用开源的 whisper.cpp 作为语音识别引擎，该引擎完全在本地运行，不涉及任何云服务或第三方数据传输。whisper.cpp 的完整源代码可在 GitHub 上获取。

隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并标注最后更新日期。
